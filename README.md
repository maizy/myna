## Developer environment

For IntelliJ Idea:
* don't forget to enable annotation processing

Requirements:
* JDK 17
* docker
* docker compose as a plugin or docker-compose utility

Start DB:

```
./dev/start-db.sh
```

Start project in dev mode:

```
./gradlew bootRun
```

Or use run configurations for IntelliJ IDEA.
